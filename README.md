## Projects

### 1) 🧮 Python CLI Calculator
A command-line calculator supporting 7 operations with full error handling and a repeat loop for continuous calculations.

**Features:**
- Addition, Subtraction, Multiplication, Division
- Power, Root (any degree), and Modulus
- Division and modulus by zero protection
- Negative number protection for roots
- Context-specific input prompts for each operation
- Repeat loop to perform multiple calculations in one session

**Concepts used:** Functions, parameters, match/case, while loops, conditionals, f-strings, input validation

**How to run:**
  python Calculator.py
